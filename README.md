# nla

<!-- badges: start -->

[![R-CMD-check](https://github.com/fwimp/nla/actions/workflows/R-CMD-check.yaml/badge.svg)](https://github.com/fwimp/nla/actions/workflows/R-CMD-check.yaml)

NLA provides a simple set of functions to aid in performing non-linear
averaging.

$$
\overline{f(X)} \neq f(\overline{X})
$$

## Installation

You can install the development version of `nla` from
[GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("fwimp/nla")
```
