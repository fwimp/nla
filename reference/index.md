# Package index

## All functions

- [`dt_modifiedlp()`](https://fwimp.github.io/nla/reference/dt_modifiedlp.md)
  : Generate a daily temperature matrix based on a modified version of
  the Logan-Parton model.
- [`dt_sin()`](https://fwimp.github.io/nla/reference/dt_sin.md) :
  Generate a simple daily temperature sine curve
- [`dt_tri()`](https://fwimp.github.io/nla/reference/dt_tri.md) :
  Generate a simple daily temperature triangle curve
- [`nla_tpc_analytic()`](https://fwimp.github.io/nla/reference/nla_tpc_analytic.md)
  : Run rate summation on a TPC with a functional form
- [`nla_tpc_empirical()`](https://fwimp.github.io/nla/reference/nla_tpc_empirical.md)
  : Perform nonlinear averaging of a temperature-dependent variable
  across a range of temperatures from empirically-derived data
